# 学习方法决策速查表

> 来源：《上瘾式学习：科学的自学成才法》— 安川康介
> 用途：遇到学习场景时直接查「当 X 时，做 Y」，不需要重读概念页

---

## 核心决策规则

### 方法选择：当你不确定用哪种方法时

| 情境 | 用什么 | 不要用什么 | 理由 |
|------|--------|-----------|------|
| 需要长期记住某个知识点 | **主动回忆**（白纸写出） | 重读 / 高亮 / 抄笔记 | 重读产生熟练度错觉，不触发测试效应 |
| 考试 / 截止日期在2周以上 | **分散学习**（多次间隔） | 集中突击 | 总时间相同时，分散学习记忆持久度高2倍 |
| 有多个相似主题要学 | **交叉学习**（打乱顺序） | 逐章分块练习 | 考试从不标注「这是第几章的知识」 |
| 内容枯燥、难以记住 | **精细提问**（追问为什么） | 强行重复 | 因果链条比事实本身更耐遗忘 |
| 特别难记的数字/术语/序列 | **记忆术**（图像转换/故事/地点） | 反复朗读 | 人脑对视觉图像的记忆能力远优于文字 |
| 学习状态差、注意力散漫 | **先运动10-20分钟** | 强行开始 | 运动激活注意力，净学习效率更高 |
| 不知道哪科要多花时间 | **元认知标注（√△○系统）** | 凭感觉分配时间 | 主观「感觉哪科弱」和客观掌握度常常不一致 |

## 方法组合优先级

```
最强组合（邓洛斯基评级双高）：
主动回忆 + 分散学习 = 连续再学习（Successive Relearning）

次强组合：
交叉学习 + 主动回忆 = 打乱题型后白纸作答

辅助组合（用于特定内容）：
记忆术 + 主动回忆 = 先建图像，再回忆图像
精细提问 + 分散学习 = 带「为什么」的间隔复习
```

## 阶段路由：根据学习阶段选方法

| 阶段 | 行动 | 不适合的方法 |
|------|------|------------|
| **完全陌生的新领域** | 先分块学习建立基础，再引入交叉 | 一开始就交叉（基础不够时适得其反） |
| **有一定基础，准备深化** | 主动回忆 + 交叉学习 | 重读（此阶段最浪费时间） |
| **复习阶段（考前2-4周）** | 连续再学习 + 混合题库 | 集中突击（对长期记忆无效） |
| **有特别难记的内容** | 先记忆术建图像，再主动回忆图像 | 直接放弃这部分内容 |
| **状态低迷时** | 运动 + 小目标切割 | 强行长时间学习（效率趋近零） |

## 关键阈值与默认值

| 参数 | 推荐值 | 来源 |
|------|--------|------|
| 睡眠时间 | ≥7小时，不可压缩 | 睡眠不足导致记忆巩固失败 |
| 运动时长 | 10-20分钟轻度即可 | 过长消耗体力，反而影响学习 |
| 每次学习后主动回忆次数 | 1-3次（直到能全部写出） | 少于1次等于没有主动回忆 |
| 单科连续学习时间上限 | 参考自身注意力极限，一般60-90分钟 | 超过后效率递减 |
| 复习间隔（7天后考试） | 3天后复习一次 | 塞佩达2008研究 |
| 复习间隔（35天后考试） | 8天后复习一次 | 塞佩达2008研究 |
| 精细提问时间投入 | 约为普通学习的1.3-2倍 | 短期慢，长期省 |

## Tells & Smells（快速识别陷阱场景）

| 你有这个感觉 | 实际情况 | 应该做什么 |
|------------|---------|-----------|
| 「重读一遍，感觉记住了」 | 熟练度错觉，不是真记住 | 立刻合书，白纸写出 |
| 「这题我一看就知道答案」 | 可能是识别而非真正回忆 | 遮住答案，先自己写，再核对 |
| 「分块学完一章，正确率90%」 | 你只是在做「分类已知的题」 | 引入其他章节题目混合做 |
| 「昨天刚学的，今天不用复习」 | 遗忘曲线在24小时内最陡 | 至少做一次短暂回忆 |
| 「考前熬夜多学几小时」 | 睡眠缩短 → 记忆巩固失败 → 净收益为负 | 正常睡眠，考前状态更重要 |
| 「这个内容太无聊，记不住」 | 缺乏情绪激活 | 追问「为什么」或用记忆术加图像 |
| 「感觉背了很多次了，应该记住了」 | 未必——主观感受和实际记忆高度不相关 | 白纸测试才是唯一可信的验证 |

## 方法适用边界（什么时候不适用）

| 方法 | 不适用的场景 |
|------|------------|
| 交叉学习 | 内容完全不同学科（如印度尼西亚语+解剖学）；完全初学时 |
| 精细提问 | 完全没有基础知识的领域（没有旧知识可连接） |
| 记忆术 | 抽象概念（难以图像化）；只需理解不需记忆的内容 |
| 分散学习 | 考试在24小时内（时间不够分散） |
| 主动回忆 | 需要深度理解而非记忆的推导类内容（虽然回忆有帮助，但不是唯一策略） |
